## Reproducibility

Download this repository

```{bash}
git clone https://github.com/research-licit/ITSProjects.git
```

Be sure to get [conda](https://www.anaconda.com/distribution/), then:

```{bash}
conda env create -f environment.yml
conda activate ITSTools
```
